# Markdown的基础语法
\#表示标题，
# 一级标题
## 二级标题，标题逐渐变小

----
### Markdown 列表的表达
#### 无序的list，列表，使用* 作为前缀
例如：
* 列表1
* 列表2

#### 对于数字列表的表达方法
1. 数字列表1
2.数字列表2
3.数字列表3

### 区块引用的写法，使用\>符号
> 第一层的
>   >  第二层的
* 注意可以叠加；
> 区块引用只需在每行的开头增加 \>
> 就可以实现了；

### 应用分割线
---
***
\---和\***都是可以作为分割线的。

### 创建连接的方式
使用\[提示语]和\(URL地址)的方式创建；

### 代码框的写法，使用\`，或者三个\`组成的代码块
` this is a code line.`
使用一个代码块
```
   public static void main(String [] args){
   }
   ```
### 重点加强主题，例如：加粗、斜体、删除线
使用\*\*加粗字体\*\*就可以实现
* ** 加粗字体 **
同理，使用\_斜体字体\_
* _斜体字体_
